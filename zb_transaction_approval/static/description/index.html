Transaction Approval

Overview

Introducing ZestyBeanz's Transaction Approval module: a powerful Odoo addon designed to manage Approval functionality in Sales Order, Purchase Order, Accounting Entries, Payments.

Key Features

Comprehensive Control: Effectively manage and control the approval functionality for the following cases.

1. Sale Orders
2. Purchase Orders
3. Invoices
4. Payments

Email Notifications: Automatically notify approvers when their approval is required.

Role-Based Access Control: Ensure that only authorized personnel can approve transactions by implementing role-based access controls. Specify which users or user groups have the authority to approve different types of transactions.

After Installing the module

Navigate to Settings and select a user.

Here added new user group section Approvers

  • Account Entries Approver
  • Payment Approver
  • Purchase Approver
  • Sale Approver

    • Working - Sale Orders

    On creating a sale order, A button 'Send For Approval' is added here.

    While clicking the button an Approval request mail send to 'sale Approver'group users.

    Sale Order Approval Request Mail

    Approve Button for approving the sale order.

    Approve Button only visible for Sale Approver Users.

    After approving the sale order, We can confirm the sale order.

    "Approved by"and "Approved On" fields available in the "Other Info" tab for tracking:

    Working - Purchase Orders

    On creating a Purchase order, A button 'Send For Approval' is added here.

    While clicking the button an Approval request mail send to 'Purchase Approver'group users.

    Purchase Order Approval Request Mail

    Approve Button for approving the Purchase order.

    Approve Button only visible for Purchase Approver Users.

    After approving the Purchase order, We can confirm the purchase order.

    "Approved by" and "Approved On" fields available in the "Other Information" tab for tracking:


    The same workflows are applied to the 'Invoices' and 'Payments' as well.
